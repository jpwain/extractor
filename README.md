# Extractor

JavaScript bookmarklet for extracting images from a web page.

To install, copy the contents of `bookmarklet.txt` to your clipboard, and create a new bookmark in your browser. Set the URL of the bookmark to the contents of the clipboard.

To build the minified version of the bookmarklet, run `python3 build.py`. This will create a new file called `bookmarklet.txt`. Optionally, pass the `-c` flag to copy the minified version to your clipboard instead of writing to a file.
